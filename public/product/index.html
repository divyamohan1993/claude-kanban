Security-First

Enterprise authentication, zero compromises

First-run setup wizard. Role-based access. Four SSO providers. Every route protected. Every action audit-logged.

Authentication Built-in

  • ✓ First-run setup wizard creates superadmin
  • ✓ Role hierarchy: superadmin > admin > user
  • ✓ Argon2id password hashing (quantum-safe)
  • ✓ AES-256-GCM field-level encryption
  • ✓ JWT sessions with configurable TTL
  • ✓ Rate-limited auth: 5 burst, exponential backoff
  • ✓ Setup permanently locked after first run

SSO Providers Enterprise

  • ✓ Built-in: local username/password
  • ✓ OIDC: OpenID Connect (Google, Okta, Auth0)
  • ✓ SAML 2.0: enterprise identity providers
  • ✓ LDAP: Active Directory integration
  • ✓ Provider secrets encrypted at rest
  • ✓ Configurable via control panel
  • ✓ Modular: swap providers without code changes

Transport Security Hardened

  • ✓ CSP with nonce-based script policy
  • ✓ HSTS, X-Frame-Options DENY
  • ✓ CSRF: Origin check + X-Requested-With
  • ✓ Token bucket rate limiting (200/100 general)
  • ✓ SSE guard: 5 per IP, 200 global max
  • ✓ Request timeouts (30s headers, 120s body)
  • ✓ Correlation ID on every request

Data Protection Defense-in-Depth

  • ✓ Path traversal prevention on all file ops
  • ✓ Parameterized SQL queries only
  • ✓ No innerHTML anywhere (XSS-safe DOM)
  • ✓ Log type allowlist (no path injection)
  • ✓ Full audit trail on all privileged ops
  • ✓ Backup retention: hot, hourly (7d), daily (30d)
  • ✓ Graceful shutdown: drain, close DB, clear PID

Key Architectural Decisions

Server-Driven UI

Backend computes available actions per card. Frontend renders buttons, zero decision-making. One source of truth.

Real-Time SSE

Server-Sent Events for live card updates, log streaming, and admin broadcasts. No WebSocket complexity.

Pipeline Lock

activeBuilds map held through full review/fix/approve cycle. Prevents race conditions across the pipeline.

Crash Resilience

Heartbeat every 30s. Uncaught exceptions logged and survived (up to 10). Orphaned cards recovered on restart.

Quick Start

Running in 30 seconds

Node 18+, pnpm, and a Claude Max subscription. That's all you need.

terminal
# Clone
$ git clone https://github.com/divyamohan1993/claude-kanban.git
$ cd claude-kanban
# Install (4 dependencies)
$ pnpm install
# Start
$ pnpm start
Claude Kanban Board http://localhost:51777 (public)
Control Panel http://localhost:51777/<random-path>
✓ Board opens in your browser. Complete setup wizard. Start building.

Single-project mode: Set KANBAN_MODE=single-project and point it at your repo. Auto-discovery scans every 30 minutes through 10 strategic lenses, creating improvement cards autonomously.
